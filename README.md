# 🌸 Iris Data Visualization

## 📌 Project Overview

This project is a part of the **Synent Technologies Data Science Internship**. The objective is to perform Exploratory Data Analysis (EDA) and Data Visualization on the Iris dataset using Python. Various visualization techniques are used to understand the distribution of features, relationships among variables, and differences between Iris flower species.

---

## 🎯 Objective

- Understand the Iris dataset
- Perform data cleaning and preprocessing
- Visualize data using different charts
- Identify patterns and relationships among features
- Draw meaningful insights from the dataset

---

## 📂 Dataset

**Dataset Name:** Iris Dataset

The dataset contains measurements of Iris flowers from three different species:

- Iris Setosa
- Iris Versicolor
- Iris Virginica

### Features

- Sepal Length
- Sepal Width
- Petal Length
- Petal Width
- Species

---

## 🛠️ Technologies Used

- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn

---

## 📊 Visualizations Performed

- Dataset Overview
- Missing Value Analysis
- Duplicate Value Analysis
- Bar Chart
- Histogram
- Scatter Plot
- Box Plot
- Pair Plot
- Correlation Heatmap

---

## 📈 Key Insights

- The dataset contains 150 flower samples.
- No missing values were found.
- The dataset is balanced across all three species.
- Setosa flowers are clearly distinguishable based on petal measurements.
- Petal Length and Petal Width show a strong positive correlation.
- Petal measurements separate species better than sepal measurements.
